# 🚀 "即刻"AI Agent助理应用 - 项目概览

## 📦 包含内容

这是一个完整的Flutter项目部署包，包含了"即刻"AI Agent助理应用的所有源代码、配置文件和文档。

## 🎯 应用功能

### 核心AI功能
- 🤖 **AI对话引擎** - 多供应商支持，3秒内首字响应
- 🎤 **语音识别** - 多API集成，实时语音转文字
- 👁️ **OCR识别** - 屏幕文字智能识别，多供应商OCR

### 用户界面
- 🪟 **悬浮窗系统** - 可调整大小和位置的智能助手窗口
- 🎨 **豆包+Claude风格** - 现代化UI设计
- 📱 **响应式设计** - 适配不同屏幕尺寸

### 系统服务
- ⚡ **后台常驻服务** - 应用可在后台持续运行
- 🔊 **唤醒词检测** - 自定义唤醒词激活
- 🔐 **权限管理** - 完善的权限申请和管理

### 安全配置
- 🛡️ **API密钥安全存储** - AES加密存储敏感信息
- 📋 **配置导入导出** - 支持配置的备份和恢复
- ⚖️ **多供应商管理** - 智能负载均衡和故障转移

## 🛠️ 快速开始

### 1. 环境要求
- **Flutter SDK** (3.16.0+)
- **Android SDK** (API Level 21+)
- **Java JDK** (8+)
- **8GB+ RAM**

### 2. 安装环境
请参考 `环境安装指南.md` 文件中的详细安装步骤。

### 3. 构建APK

#### 方式一：自动化构建 (推荐)
```bash
chmod +x auto_build.sh
./auto_build.sh
```

#### 方式二：分步构建
```bash
cd jike_ai_agent
flutter clean
flutter pub get
flutter build apk --release
```
